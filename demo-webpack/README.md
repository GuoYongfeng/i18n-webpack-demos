# i18n-webpack-demos

基于webpack的国际化方案探索

## 实现特性

- 根据locals语言包实现四种语种编译
- 实现多js入口资源编译
- 实现多模板进行资源编译
- 支持less语法解析
- 支持EJS和HTML模板编译
- 支持文件hash指纹

## 如何使用该项目

```
$ npm install
$ npm start
```
